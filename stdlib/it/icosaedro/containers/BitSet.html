Set of non-negative integer numbers

Version: $Date: 2012/02/22 10:41:48 $

Author: Umberto Salsi <salsi@icosaedro.it>

The size of the set grows dynamically as more elements are entered. Bits have a position given by an index that starts from 0 (the first bit). The set is open to the right, and bits above the last bit set or cleared are assumed zero. Example: 

     /**
      * Eratosthenes sieve algorithm.
      * @param int $n Search prime numbers up to this limit.
      * @return BitSet Found prime numbers.
      */
     function primes($n)
     {
         $b = new BitSet();
         for($i = 2; $i <= $n; $i++)
             $b->set($i);
         $i = 2;
         while( $i * $i <= $n ){
             if( $b->get($i) ){
                 $k = 2 * $i;
                 while( $k <= $n ){
                     $b->clear($k);
                     $k += $i;
                 }
             }
             $i++;
         }
         return $b;
     }

     echo primes(100);
