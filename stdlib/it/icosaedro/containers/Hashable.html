int getHash()

Returns the hash of the object

The hash value must be consistent with the equality test, that is if two distinct objects carries the same value, then the resulting hash must be the same: if $a->equals($b) then $a->getHash() === $b->getHash() must be true as well. Also remember that if two objects result to be equal to each other, then the hash value must be the same. Moreover, the objects, once defined, must be immutable, so that the hash of a given object never change.

Return: The hash.
