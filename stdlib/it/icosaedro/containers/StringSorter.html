interface it\icosaedro\containers\StringSorter

Generic interface that defines a sort criteria among strings

Version: $Date: 2012/01/28 20:27:32 $

Author: Umberto Salsi <salsi@icosaedro.it>

For example, you may define a custom comparator that depends on a particular encoding of the strings, or you define a case-insensitive comparator, a phone book sort, etc. Custom comparators can be used with the sortArrayOfStringByComparator() method of the Arrays class.
