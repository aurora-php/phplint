interface it\icosaedro\regex\UGroup

Interface to access a matched group of elements in a regular expression

Version: $Date: 2012/03/17 17:18:37 $

Author: Umberto Salsi <salsi@icosaedro.it>

A group is an element possibly followed by a quantifier, for example (X)*+. Since the element (X) may match several times, this interface allows to access every matched occurrence of the element. The count() method returns the number of times the element matched.
