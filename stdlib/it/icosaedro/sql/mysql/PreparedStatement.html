Sets a parameter of type binary (for example, an image)

This implementation is specific of MySQL and assumes a field of type VARBINARY. Please note that the BINARY type should not be used because the fields of this type are automatically right-padded with spaces or NUL bytes depending on the version of MySQL, which normally is not what you want.

Parameters:
$index   Index of the parameter to set, the first question mark is the number 0.
$value   Value of the parameter as an array of bytes. If NULL, then the SQL NULL value is set.

Throws:
  • it\icosaedro\sql\SQLException    If the index is invalid.
