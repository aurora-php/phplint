interface it\icosaedro\sql\SQLDriverInterface

Minimalistic, generic SQL driver interface

Version: $Date: 2012/02/20 18:17:52 $

Author: Umberto Salsi <salsi@icosaedro.it>

Every implementation should specify how strings of characters are encoded (for example, UTF-8) and then all the strings that appear in this class and its implementations are assumed to work with that same encoding. Example: 

	$db = new \it\icosaedro\sql\mysql\Driver( array("localhost", "myname", "mypass", "test") );
	$rs = $db->query("select current_date");
	$rs->moveToRow(0);
	echo "Date: ", $rs->getDateByIndex(0), "\n";
