Evaluate basic statistical parameters about a set of samples

Version: $Date: 2012/02/09 10:14:23 $

Author: Umberto Salsi <salsi@icosaedro.it>

The initial set of samples is empty. Example: 

	$s = new Statistics1D();
	$s->put(1.0);
	$s->put(2.0);
	$s->put(3.0);
	echo $s;
displays: 
	n=3 min=1 max=3 mean=2 dev=1

Remember that if you generate an uniform distribution of values in the range [a,b], then the expected mean is (a+b)/2 and the standard deviation is (b-a)/sqrt(12) = 0.29 * (b-a).
