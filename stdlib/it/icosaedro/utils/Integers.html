static int magnitude(
        int $i)

Returns the magnitude of the int as unsigned number

The magnitude is the number of bits remaining in the machine word once all the leading zeroes have been stripped off. Then, the magnitude of 1 is 1; the magnitude of 0 is 0; the magnitude of -1 is 8 * PHP_INT_SIZE.

Parameters:
$i   Any int number.

Return: Magnitude of the int as unsigned number.
