standard_reflection

Reflection

PHP version: 5

Required modules: standard

Required packages: none

Actually this module should be part of the standard module, and it is available on any PHP 5 installation. I made a separated one for performance reasons, since it is seldom used.

See: www.php.net/manual/en/book.reflection.php
