PHP version: 5

Required modules: none

Required packages: none

Deprecated: Very poorly written interface with too many 'mixed' values, difficult to validate automatically, leaving to the programmer all the responsability to handle properly arguments and returned values. The returned values are also 'mixed' with a complex semantic based on the special values NULL, FALSE/TRUE, etc. It is questionable if this library really adds security. [Umberto Salsi]

See: www.php.net/manual/en/ref.filter.php.
